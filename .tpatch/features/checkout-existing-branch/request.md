This enables migrating existing in-progress work into the tesseraworkspaces model.

## Description

Allow tws new to check out an existing branch into a feature worktree instead of only creating new branches. Should work even if the branch is currently checked out in the main repo (git worktree add --detach the main repo first, or use a different approach). This enables migrating existing in-progress work into the tesseraworkspaces model.

# Feature Request: Add a tws archive command that removes the worktree from disk but keeps the branch reference in stack.yaml. This saves disk space for large repos where worktrees accumulate. A tws restore or tws new with the same branch name should be able to recreate the worktree from the archived branch. Long-term quality of life improvement.

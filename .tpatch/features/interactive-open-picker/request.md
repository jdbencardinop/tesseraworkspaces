Could also support tws open <feature> with no branch to pick from branches within that feature. Shell completions from cobra should also suggest valid features and branches.

## Description

When tws open is called without arguments, show an interactive picker (via fzf or built-in) to select a feature and branch. Could also support tws open <feature> with no branch to pick from branches within that feature. Shell completions from cobra should also suggest valid features and branches.
